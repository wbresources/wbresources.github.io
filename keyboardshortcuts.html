WB Label Generator -- Keyboard Shortcuts


Here is a list of keyboard shortcuts that you can use on this website to make generating/copying labels much faster:


    Label Shortcuts

  • { ENTER } - Copy label

  • { D } - Copy a normal/digital Daily Planner label with today's date for whatever grade and gender you selected

  • { C } - Change type to CW and copy label

  • { H } - Change type to HW and copy label


    • Theme Shortcuts

  • { 1 } - Change the theme to home

  • { 2 } - Change the theme to light_mode

  • { 3 } - Change the theme to rocket_launch

  • { 4 } - Change the theme to local_florist

  • { 5 } - Change the theme to directions_car
